# geoffont.github.io

Portfolio personnel de Geoffroy, développeur fullstack basé à Marseille.

## Stack

- HTML, CSS, JavaScript (vanilla)
- Google Fonts (DM Serif Display, Outfit, JetBrains Mono)
- GitHub Pages

## Structure

```
.
├── index.html        # Page principale
├── css/
│   └── styles.css    # Styles
└── js/
    └── main.js       # Animations scroll & nav
```

## Lancer en local

Ouvrir `index.html` dans un navigateur ou utiliser un serveur local :

```bash
npx serve .
```

## Déploiement

Push sur `main` — GitHub Pages déploie automatiquement.
